# ONNX Checker

## About

ONNX Checker is a simple tool to check whether onnx file can be imported in Unity/Barracuda without Unity Editor.  

## How to Use

```
onnx_checker.exe ./model.onnx
```

## Environment

* Unity 2020.3 LTS
* Barracuda 3.0.0
